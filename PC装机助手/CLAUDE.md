# CLAUDE.md — PC 装机助手

> 完整需求规格见 `提示词_需求规格.md`。本项目是该规格的实现。

## 项目定位

单页 Web 应用（`pc-builder.html`），帮助用户：
1. 逐项选择电脑配件并自动检查兼容性
2. 评估整机游戏性能（7 款游戏 × 3 分辨率）
3. 评估 AI 推理能力（可部署模型 + token 速度）

## 技术约束

- **单文件交付**：所有 HTML/CSS/JS 内嵌于 `pc-builder.html`
- **零外部依赖**：不引入任何 CDN、npm 包、框架
- **打开方式**：浏览器直接双击即可使用
- **暗色主题**：GitHub dark 风格配色

## Loop Engineering 工作流

### 检查清单（Checker 必查）

对 `pc-builder.html` 的检查分为 6 类：

**1. HTML 结构检查**
- `<!DOCTYPE html>` 声明存在
- `<meta charset="UTF-8">` 存在
- 所有标签正确闭合
- 左侧 8 个配件板块 + 右侧 2 个 Tab 都有对应 DOM 结构

**2. CSS 样式检查**
- 暗色主题配色正确（背景 `#0d1117`，面板 `#161b22`）
- 左侧边栏宽度 320px，右侧自适应
- 折叠/展开动效正常
- 响应式断点存在（≥1366px 桌面端优先）

**3. 数据完整性检查**
- CPU 数据 ≥ 12 款（Intel 6+ / AMD 6+）
- GPU 数据 ≥ 12 款（NVIDIA 8+ / AMD 4+），每款含显存+位宽+带宽
- 主板数据 ≥ 10 款，含插槽/芯片组/内存类型/供电等级
- 内存 ≥ 8 款（DDR4+DDR5）
- 其他配件各 ≥ 4 款
- 所有价格贴近 2025 年中国市场行情

**4. 兼容性逻辑检查（核心）**
- CPU Intel → 主板仅显示 LGA1700 芯片组
- CPU AMD AM5 → 主板仅显示 AM5 芯片组
- CPU AMD AM4 → 主板仅显示 AM4 芯片组
- i9/R9 高功耗 CPU → 过滤掉 H610/A620 等丐版
- 选择 DDR5 主板 → 内存仅显示 DDR5
- 选择 DDR4 主板 → 内存仅显示 DDR4
- CPU+GPU 功耗计算 → 电源推荐瓦数正确
- 上游选择变化 → 下游自动清空并重新过滤

**5. 性能评估检查**
- 7 款游戏 × 3 分辨率 = 21 个基准帧率数据（不同 GPU 有差异）
- GPU 切换后所有帧率数据同步更新
- 体验评级颜色正确（≥144 金色、≥90 绿色、≥60 浅绿、≥45 黄色、≥30 橙色、<30 红色）
- AI 模块：根据显存推荐模型列表，显示 token 速度

**6. 交互体验检查**
- 板块折叠/展开正常
- 品牌切换过滤正常
- 价位过滤正常
- 已选配件高亮显示
- 总价实时计算正确
- 不兼容项灰色+tooltip 提示

### 停止条件

1. **ALL GREEN**：所有 6 类检查通过，附每项通过证明
2. **同一失败连续 2 轮**：升级并报告
3. **回归**：修复导致之前通过项失败
4. **轮次用尽**：最多 5 轮

### 红线

- 永不弱化/跳过兼容性检查逻辑
- 永不为"好看"而编造性能数据（基准帧率参考真实评测）
- 永不删除已有正常功能

## 命令

| 用途 | 命令 |
|------|------|
| 语法检查 | 用浏览器打开 `pc-builder.html`，F12 Console 检查无 JS 报错 |
| 兼容性测试 | 选 i5-13600KF → 确认主板只显示 LGA1700 → 选 Z790 DDR5 → 确认内存只显示 DDR5 |
| 基准数据验证 | 选 RTX 4090 → 确认赛博朋克 4K 帧率 ≈ 80-90 fps（真实评测范围） |
| 总价验证 | 全选最便宜配件 → 总价 ≈ 4000-5000 元；全选旗舰 → 总价 ≈ 25000-35000 元 |
